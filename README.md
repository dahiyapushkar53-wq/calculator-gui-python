# 🧮 Calcify GUI – Python Calculator

## 📌 Description

**Calcify GUI** is a simple and interactive calculator application built using **Python** and the **Tkinter library**.
This project provides a clean graphical user interface (GUI) that allows users to perform basic arithmetic operations easily.

It is designed as a **beginner-friendly project** to demonstrate concepts of GUI development and event-driven programming in Python.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📟 Real-time display of input and result
* 🧼 Clear (C) button to reset calculations
* 🖱️ Button-based interactive interface

---

## 🛠️ Technologies Used

* **Python 3**
* **Tkinter (Standard GUI Library in Python)**

## ▶️ How to Run the Project

### Step 1: Install Python

Make sure Python (version 3.x) is installed on your system.

### Step 2: Clone the Repository

```bash
git clone https://github.com/your-username/calcify-gui.git
```

### Step 3: Navigate to Project Folder

```bash
cd calcify-gui
```

### Step 4: Run the Program

```bash
python main.py
```
